## Descrição
Aplicação de console em C# que realiza conversão entre Celsius, Fahrenheit e Kelvin.

## Funcionalidades
- Celsius para Fahrenheit
- Fahrenheit para Celsius
- Celsius para Kelvin

## Tecnologias
- C# 
- .NET
